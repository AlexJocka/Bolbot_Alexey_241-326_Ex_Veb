## Описание

Веб-приложение разработано на Flask для автоматизации работы электронной библиотеки.

Реализовано:

- просмотр списка книг;
- просмотр подробной информации о книге;
- добавление, редактирование и удаление книг;
- загрузка обложек книг;
- хранение жанров и связь книг с жанрами;
- авторизация пользователей;
- разграничение прав по ролям: администратор, модератор, пользователь;
- добавление рецензий к книгам;
- подсчёт средней оценки и количества рецензий;
- пагинация списка книг;
- flash-сообщения;
- поддержка Markdown в описании книг и рецензиях;
- очистка введённого текста через Bleach.

## Индивидуальный вариант 3

Добавлена функциональность поиска книг на главной странице.

Поиск выполняется по следующим полям:

- название;
- жанр;
- год;
- объём страниц от/до;
- автор.

Для названия и автора используется частичное совпадение. Если поле поиска не заполнено, оно не учитывается при фильтрации. При переходе между страницами параметры поиска сохраняются.

## Локальный запуск

```bash
python3.11 -m venv ve311
source ve311/bin/activate
pip install -r requirements.txt
cd app
python init_db.py
flask --app app run
```

После запуска приложение доступно по адресу:

```text
http://127.0.0.1:5000
```
